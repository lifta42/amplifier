Detailed Description

A naive implementation of key-value dictionary. For more info, check notes on naive dictionary.

Author
liftA42
Date
Dec 12, 2017

The dictionary uses linear search on small data set, and hash table on bigger one.

A verb for hashing the keys and another verb for checking if two keys are equal are required.
